# SE330 Airline Booking Desktop App

This repository captures the project scope, architecture, and UI/UX direction for the SE330 airline booking desktop application (Swing + JDBC + PostgreSQL). The detailed agent-ready specification is provided in `docs/ai-agent-dev-spec.md`.

## Contents
- **Project scope & requirements**: SE330 academic goals + CNPM process goals
- **Architecture**: Swing client, JDBC services/DAO, PostgreSQL VPS, CI pipeline
- **Functional scope**: User/Admin MVP features and enhancements
- **Non-functional requirements**: anti-oversell, idempotency, security, audit logging
- **Theme system**: Light/Dark tokens and Swing theming guidance
- **UI/UX direction**: Modern Airline Minimal + Clean Dashboard
